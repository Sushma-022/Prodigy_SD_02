# Number Guessing Game 🎯

This is my submission for **Task 2** of the Prodigy InfoTech Software Development Internship.

## 📝 Description
A Python command-line interactive game where the computer generates a random number between 1 and 100, and the user must guess it. The program provides continuous feedback (too high or too low) and tracks the total number of attempts it takes for the user to win the game.
